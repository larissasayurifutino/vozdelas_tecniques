Objetivo

Monitorar em tempo real os textos da Folha identificando e quantificando mulheres entrevistadas.
Com isso, determinar áreas em que mais mulheres podem/devem ser mais ouvidas/citadas.
Por fim, ter um banco de referências de mulheres ouvidas disponível para sugerir referências para os jornalistas da Folha.

De modo geral:

Lemos as reportagens e as representamos a partir do seu conteúdo textual.
Reconhecemos quais trechos de texto são nomes próprios.
Atribuímos um gênero para cada um dos nomes reconhecidos nos textos.
Identificamos quais trechos de texto são citações diretas.
Definimos quem foi o autor de cada uma das citações.

01) Ler Textos

Consideramos que as reportagens estão disponíveis em arquivos.

Cada reportagem é representada pelo seu conteúdo textual: título, sub-título e o texto propriamente dito.

02) Processamento de Texto

Essa é a etapa de manipulação dos dados que temos.

Textos como dados é algo relativamente recente mas temos várias ferramentas específicas para o Português Brasileiro.

Usamos as implementações do pacote spacy da linguagem Python.

02.A) Reconhecimento de entidades

Essa é a etapa que equivale a reconhecer quais trechos de texto são nomes próprios.

Cada texto passa por um modelo de Reconhecimento de Entidades (named entity recognition, ou NER em inglês).

O modelo reconhece trechos de texto que constituem nomes próprios e categoriza entre Pessoa, Localização, Organização, Moeda, por exemplo.

Filtramos para manter apenas entidades do tipo Pessoa.

02.B) Reconhecimento de citações

Em cada texto reconhecemos os trechos que são citações diretas de uma pessoa.

Para isso, usamos algo mais simples: Expressões Regulares.

Logo, estamos identificando um padrão de texto com uma linguagem específica para isso.

02.D) Atribuir gênero para os nomes encontrados

Cada entidade é representada pelo seu 1º nome próprio.

Usamos uma base do IBGE para atribuir um gênero a cada nome, disponível em: brasil.io/dataset/genero-nomes.

02.E) Atribuir cada citação a um nome próprio

Precisamos responder: Dadas essas X declarações e essas pessoas Y quem falou o quê?

Parece intuitivo que uma declaração seja acompanhada de quem a fez.

02.F) Caracterizar entidades

Quem são as mulheres entrevistadas?

Pegamos os termos que acompanham (estão na vizinhança) das entidades.

Desses termos ficamos com os substantivos e adjetivos que são do mesmo gênero da entidade citada.

03) Métricas

Cada texto é representado por métricas resumo que são basicamente contagens de entidades com e sem citação por gênero.

04) Resultados

As métricas são acompanhadas diariamente e são reportadas aos jornalistas e editorias por meio de e-mail semanal.
